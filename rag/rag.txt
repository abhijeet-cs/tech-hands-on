📄 Step 1: Document & Pre-processing
- Split data into chunks
- Convert chunks to embeddings (vectors)
- Store in Vector DB

🔍 Step 2: Query Processing
- Convert query to vector
- Retrieve similar chunks
- Send query + context to LLM

🆚 Vector Store vs Vector DB
- Tool vs Full database
- Local vs Enterprise
- Basic vs Advanced search (filters, metadata)
- Prototype vs Production

📏 Similarity Scores

Chroma (L2 Distance):
- Lower score = more similar
- 0 = identical

Cosine Similarity:
- Higher score = more similar
- Range: -1 to 1
